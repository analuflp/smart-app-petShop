## Smart App PetShop

Este é um projeto  mobile simples de pet shop, desenvolvido em Kotlin, que consome a API de pet shop para listar e adicionar pets. O projeto segue a arquitetura MVVM e utiliza componentes como RecyclerView para a exibição de listas e Retrofit para o consumo da API.

### Funcionalidades
- Listagem de Pets: Exibe uma lista de todos os pets cadastrados.
- Adicionar Pet: Permite adicionar um novo pet.
### Tecnologias Utilizadas
- Kotlin: Linguagem de programação utilizada no desenvolvimento do aplicativo.
- MVVM: Padrão de arquitetura utilizado para separar responsabilidades.
- Retrofit: Biblioteca utilizada para fazer as requisições HTTP para a API.
- RecyclerView: Componente utilizado para exibir a lista de pets.
- ViewModel: Componente do Jetpack utilizado para gerenciar os dados de UI de maneira cíclica.
- LiveData: Utilizado para observar mudanças nos dados e atualizar a UI automaticamente.
### Requisitos
- Android Studio (versão mais recente recomendada)
- Dispositivo ou emulador Android com API 21 ou superior
